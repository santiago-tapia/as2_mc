# as2_mc
Aerostack 2 Modular Classes

# Compiling with ninja-build

Ninja is a fast compiling builder.

## Installing:

```bash
sudo apt install ninja-build
```
## Use:

Do:

```bash
export CMAKE_GENERATOR=Ninja
```

Or add it to .bashrc:

```bash
echo "export CMAKE_GENERATOR=Ninja" >> ~/.bashrc
source ~/.bashrc
```
